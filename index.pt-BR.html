Como o nome genérico sugere, é um editor de texto com código aberto, escrito em C, que usa a bilbioteca GTK4 e é designado para Linux.

O código fonte está distibuído em vários arquivos de C contidos no diretório [source/]. O ponto de entrada do programa é o arquivo [main.c], que inclui os demais arquivos para ser compilado.

A compilação pode ser feita pelo script [build.sh], que não requer privilégio root. Basta executá-lo usando algum shell. Exemplo:

oksh build.sh

Isso resultará em um binário denominado [gte]. Para instalá-lo em definitivo no sistema, ele deve ser copiado para algúm diretório do PATH (preferencialmente, [/usr/local/bin/]). Essa parte requer privilégio:

sudo cp gte /usr/local/bin/

Cada janela contém apenas um arquivo para ser editado e não é possível passar mais de um nome de arquivo pela linha de comando para ser aberto.

gte some-file.txt

Várias instâncias do [gte] podem estar abertas ao mesmo tempo. Cada uma é isolada.
