Coisas relacionadas à cursor.

gte_cursor_set_position

Entrada:

  • int line
  • int column

Posiciona o cursor na [line] e [column], que começam a partir de 1 (não de 0) e atualiza o texto de status do cursor [gte_cursor_position_label].

Não retorna nada.

gte_cursor_menu_button_create

Não recebe entrada.

Cria:

  • [gte_cursor_menu_button]

    • Um widget do tipo [GtkMenuButton] que mostra a posição do cursor e, ao ser pressionado, exibe o popover para definí-la.

  • [gte_cursor_popover]

    • Um widget do tipo [GtkPopover] que permite que o usuário defina a posição do cursor.

Não retorna nada.
